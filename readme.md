# NetflixCrawler

NetflixCrawler is getting new contents of netflix.

## Installation

```bash
git clone https://github.com/soronto3603/NetflixCrawler.git
or
you can download as ".zip" file
```

## Usage

```bash
npm start
```

check the file "output.json".
